# 💳 Credit Card Fraud Detection Dashboard

An end-to-end fraud detection web application built using **Flask, SQLite, SQL, and Machine Learning**.

This project demonstrates a complete data pipeline:

- Data ingestion
- SQL database integration
- Exploratory analysis
- Machine learning classification
- Interactive dashboard
- Web deployment using Flask

---

## 🚀 Project Overview

Financial fraud detection is a critical problem in the banking and fintech industry.

This project builds a full-stack analytics application that:

- Loads transaction data into a SQL database
- Performs data analysis using SQL queries
- Trains a classification model to detect fraudulent transactions
- Displays insights in an interactive dashboard
- Allows users to test fraud prediction via a web interface

---

## 🏗 Architecture
CSV Dataset
↓
SQLite Database (SQL queries)
↓
Flask Backend
↓
Machine Learning Model (Logistic Regression)
↓
Interactive Dashboard (Chart.js)

Fraud Prediction Interface
